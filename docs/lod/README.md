# Lod

Create, Update, and Preset LODs of selected objects.

![](/img/lod.png)

---

## Create

- Press `Add` to add number of LODs
- Set the reduction percentage of the LODs

![](/img/lod_add.gif)

## Update

- Update number of LODs
- Change the reduction percentage of the LODs
